# JavaScript DOM Manipulation Tasks

This repository contains a series of mini-projects built to practice Document Object Model (DOM) manipulation, event handling, and dynamic UI updates using vanilla JavaScript.

### Projects
- [x] **1. Real-Time Digital Clock:** An interface that updates the time every second using DOM methods.
- [x] **2. Dice Rolling App:** Generates and displays a random number (1-6) when a button is clicked.
- [x] **3. Random Background Color:** An app that dynamically changes the webpage's background to a random color upon interaction.
- [x] **4. Daily Dua App:** Displays a different Dua dynamically each time the user clicks a button.

### Tech Stack
* **HTML5:** Page structure
* **CSS3:** Styling and layout
* **JavaScript (ES6):** DOM manipulation, Math methods, and Event Listeners.

### How to View
To test any of these applications, simply open the link in respective description section in any modern web browser.
